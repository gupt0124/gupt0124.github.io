Five HTML Tags

Blockquote
A large, stand alone quote from another source.
Citation
It is a person name, a URL, a book, a movie title, etc. for another source is used with quotations.
Emphasis
This tag changes the test to italic.
Important text
This tag is used to highlight a text as it bolds that text.
Line Break
This tag is used to put a line break between content.
